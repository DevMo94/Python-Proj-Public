# Python Utility Scripts

A collection of lightweight Python scripts designed to automate everyday tasks and improve productivity.

---

## Features

1. **[Investment Tracker](https://github.com/DevMo94/Python-Proj-Public/blob/main/Investments%20Tracker/Investments%20Tracker.py)**  
   - Automates monitoring of personal investments by running continuously during market hours. This allows real-time tracking of stock prices and easier management of investment decisions while working.
   
3. **[Network Latency Monitor](https://github.com/DevMo94/Python-Proj-Public/blob/main/Internet%20Latency%20Test/Latency%20%26%20Anomalies%20Testing.py)**  
   - Measures internet latency under heavy usage conditions. The script helps identify performance drops, enabling timely escalation to the service provider when issues arise.  

4. **[Credit Card Expense Manager](https://github.com/DevMo94/Python-Proj-Public/blob/main/PDF%20to%20Excel%20Data%20Extract/Extract%20Data%20from%20PDF%20to%20Excel.py)**  
   - Streamlines monthly expense tracking for multiple family members. By uploading a credit card statement in PDF format, the script automatically processes the data into a structured Excel file. This reduces manual effort and facilitates sending personalized summaries of due amounts to each member.

---

## Installation

Clone the repository:

```bash
git clone https://github.com/DevMo94/Python-Proj-Public.git
cd Python-Proj-Public

```

## Usage

- Investment Tracker: Run the script during market hours to continuously monitor stock prices.
- Network Latency Monitor: Execute the script during peak internet usage to log latency data.
- Credit Card Expense Manager: Upload your monthly PDF statement and run the script to generate an Excel file with formatted expense details.
